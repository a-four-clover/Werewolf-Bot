# Werewolf-Bot
Discordで人狼ゲームを自動進行するBotです。
役職のカスタムやJSONによるゲーム設定変更に対応しています。
なお、GitHubへの投稿は初めてなのでミス等あってもご容赦ください。

## Features
- 人狼ゲーム進行の自動化
- カスタム役職
- JSONによる設定変更
- Discord Embedを用いたUI

## Tech Stack
- Python
- discord.py
- pytest

## Project Structure
src/ : Bot本体  
roles/ : 役職設定  
tools/ : 補助スクリプト  

## Setup
1. Pythonをインストール
2. 依存関係をインストール
